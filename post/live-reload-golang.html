Live reload em Golang

Antes de mais nada é necessário instalar o AIR na máquina, para isso siga os seguintes passos:

  1. Rodar o seguinte comando go install github.com/cosmtrek/air@latest
  2. Com um projeto aberto, basta rodar os seguinte comando para iniciar air init
  3. Com isso irá criar os seguintes arquivos
    • tmp = Fica o executável que foi feito o build;
    • air.toml = Arquivo com as configurações para rodar o projeto;
  4. Rodar o seguinte comando air, logo depois deverá rodar o AIR nesse projeto
  5. Qualquer alteração dentro do arquivo "main.go" deverá fazer o build novamente do app
